Three Implementation Methods

onclick Handler (Simplest)

Use onclick attribute directly on buttons for quick implementation.

HTML - General Inquiry
<button onclick="openWhatsAppQuote()"> <i class="fab fa-whatsapp"></i> Get a Quote </button>
HTML - Product Inquiry
<button onclick="inquireProduct('Surgical Gloves')"> <i class="fab fa-whatsapp"></i> Inquire </button>
✅ Best for: Quick implementation, simple pages, basic functionality

Data Attributes (Recommended)

Use data attributes for scalable, maintainable code. Auto-initialized by WhatsAppManager.

HTML - General Inquiry
<button data-whatsapp-type="general"> <i class="fab fa-whatsapp"></i> Get a Quote </button>
HTML - Product Inquiry
<button data-whatsapp-type="product" data-whatsapp-product="Surgical Gloves"> <i class="fab fa-whatsapp"></i> Inquire </button>
HTML - Clickable Image
<img src="product.jpg" data-whatsapp-type="product" data-whatsapp-product="Surgical Gloves" style="cursor: pointer;">
✅ Best for: Large projects, multiple pages, easy maintenance, SEO-friendly

WhatsAppManager API (Advanced)

Direct access to WhatsAppManager object for custom implementations.

JavaScript - Direct API Calls
// General inquiry WhatsAppManager.sendGeneralInquiry(); // Product inquiry WhatsAppManager.sendProductInquiry('Product Name'); // Custom message WhatsAppManager.sendCustomMessage('Your message here'); // Build custom URL const url = WhatsAppManager.buildUrl('Hello!'); WhatsAppManager.openChat(url);
HTML - With onclick
<button onclick="WhatsAppManager.sendGeneralInquiry()"> Contact Us </button>
✅ Best for: Complex interactions, custom workflows, dynamic product data

Quick Integration Checklist

  • ✅ Load script.js at end of body: <script src="js/script.js"></script>
  • ✅ Choose one implementation method (onclick / data attributes / WhatsAppManager)
  • ✅ Add WhatsApp number to buttons or data attributes
  • ✅ Test on mobile and desktop browsers
